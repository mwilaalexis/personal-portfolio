A propos

Je m'appelle Kilolo Mwila Alexis. Je developpe des backends structures (repository pattern, injection de dependances, Entity Framework) et des interfaces React modernes.

Mon projet principal est NutriGuide : un systeme nutrition complet que j'ai decoupe en plusieurs services .NET 8, relies par une API Gateway et utilises depuis une application React.

NutriGuide - Systeme complet

Application de nutrition modulaire : l'utilisateur se connecte, consulte des aliments, construit des plans repas et suit son profil. Chaque partie du systeme a son propre depot GitHub.

Chaque service possede son README avec instructions de lancement. Le gateway centralise les appels du frontend vers les APIs backend.
